# **EBS (ELASTIC BLOCK STORE)**
EBS é um serviço da AWS que fornece discos virtuais para as suas instâncias EC2.
Pensa nele como o HD/SSD do seu servidor na nuvem.
&nbsp;
# **TIPOS DE VOLUME EBS**
- HDD (lento / +espaço) - HARD DISC DRIVE
- SSD (rápido / -espaço) - SOLID STATE DRIVE
&nbsp;
# **EFS (ELASTIC FILE SYSTEM)**
AWS EFS é um sistema de arquivos compartilhado, totalmente gerenciado, que pode ser acessado por várias instâncias EC2 ao mesmo tempo.
Pensa nele como uma pasta compartilhada na nuvem, que todos os servidores podem montar e usar juntos.
&nbsp;
# **FSx**
AWS FSx é uma família de serviços que oferece sistemas de arquivos especializados, geralmente usados em ambientes corporativos, Windows, alta performance, processamento científico etc.
Ele não é um único tipo de storage, mas sim quatro opções diferentes, cada uma para uma necessidade específica.
