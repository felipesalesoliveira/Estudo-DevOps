# **SCALE SET**
- É um recurso do Azure que permite criar e gerenciar automaticamente um grupo de VMs idênticas, com foco em alta disponibilidade e escalabilidade.
&nbsp;

# **APP SERVICE**
- Serviço de Aplicativo do Azure (Azure App Service) é um serviço PaaS para hospedar aplicações web, APIs e backends sem precisar gerenciar servidores ou VMs.
&nbsp;

# **AZURE CONTAINER INSTANCE**
- O Azure Container Instances (ACI) é um serviço do Azure para rodar containers Docker diretamente, sem precisar criar VM nem Kubernetes.
